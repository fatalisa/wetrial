## Install

```bash
npm install --save wetrial
```

专为wetrial定制的 基于ant-design的通用业务组件库，包含通用组件、通用样式、辅助方法等

完整 API 使用文档，请[这边走](https://xiexingen.github.io/wetrial)

## Usage

```jsx
// 比如
import React, { Component } from 'react'
import { Authority } from 'wetrial'

```
